GIT

Basic Git commands that I use all the time:

git init creates  a new local repository

git clone creates a copy working of a local repository

git commit -m” bla bla bla” Commits changes ti head (but not yet to the remote repository )

git commit -a Commits any files you’ve added with git add, and also commits files you’ve changed since then

git push Send changes to the master branch of your remote repository 

git status List the files you’ve  changed and those you still need add or commit
