Selenium WebDriver

High Level steps for creating your first selenium test

  • Starter project - https://github.com/dnajd/ruby_tester
  • Assertions - http://www.mattsears.com/articles/2011/12/10/minitest-quick-reference
  • WebDriver Docs - http://www.seleniumhq.org/docs/03_webdriver.jsp

Create First Test

Create folder for tests

/test/nameproject

Create a test inside the folder

/test/nameproject/login_test.rb

Copy selenium example into the test

/test/examples/selenium_test.rb

Modify selenium example code in

/test/nameproject/login_test.rb

ex code…

@driver.navigate.to "http://my.site.com" @driver.find_element(:id, “username").send_keys “kikluz”

Modify Rakefile to run the test

Run the test add it on Rekefile

ex. t.pattern = 'test/**/*login_test.rb'

Run the test in the terminal

be rake test

Extract find_element code to Page Object

Create a folder for page objects

/tests/bayphoto/page_objects/login_page.rb

class LoginPage def initialize(driver) @driver = driver end

def get_customer_number
    return @driver.find_element(:name, 'new_session[customerid]')
end

end

Use the page object in /test/bayphoto/login_test.rb

replace this…

@driver.navigate.to "http://my.bayphoto.com" @driver.find_element(:id, “username’’).send_keys “kikluz”

with something like this…

@driver.navigate.to "http://my.bayphoto.com" login_page = LoginPage.new(@driver) login_page.get_customer_number.send_keys "kikluz"
